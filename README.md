## MQ9 / SEN0134 — Μονοξείδιο του Άνθρακα (CO)

### Τεχνικά Χαρακτηριστικά
| Παράμετρος | Τιμή |
|---|---|
| Τύπος εξόδου | Αναλογική τάση |
| Εύρος εξόδου | 0 – 5 V |
| Εύρος ανίχνευσης CO | 10 – 1.000 ppm |
| Load resistor (RL) | 10 kΩ (στη μονάδα DFRobot) |
| Τροφοδοσία | 5 V |
| Χρόνος προθέρμανσης | >24 ώρες |

### Ανάλυση Datasheet
Ο MQ9 είναι **χημειοαντιστατικός** αισθητήρας: η εσωτερική του αντίσταση (Rs)
μειώνεται καθώς αυξάνεται η συγκέντρωση CO.

#### Βήμα 1 — Υπολογισμός Rs από την τάση
Ο αισθητήρας και ο RL σχηματίζουν έναν διαιρέτη τάσης:

```
Vout = Vc × RL ÷ (Rs + RL)
```

Λύνουμε ως προς Rs:

```
Rs = RL × (Vc - Vout) ÷ Vout
   = 10 kΩ × (5,0 - Vout) ÷ Vout
```

#### Βήμα 2 — Υπολογισμός λόγου Rs/R0
Το **R0** είναι η αντίσταση του αισθητήρα σε **καθαρό αέρα**.
Από το datasheet: `Rs/R0 = 9,9` σε καθαρό αέρα.

```
R0 = Rs_καθαρός_αέρας ÷ 9,9
```

#### Βήμα 3 — Μετατροπή σε ppm (καμπύλη CO από datasheet)
Το datasheet δίνει log-log γράφημα Rs/R0 vs ppm.
Η εξίσωση της καμπύλης για **CO**:

```
ppm = 100 × (Rs/R0) ^ -1.53
```

### Πλήρης Τύπος
```cpp
float v     = analogRead(pin) * (5.0 / 1023.0);
float Rs    = 10.0 * (5.0 - v) / v;          // kΩ
float ratio = Rs / R0;
float ppm   = 100.0 * pow(ratio, -1.53);
```

### Απλοποιημένος Τύπος (για μαθητές)
```
ppm ≈ analogRead × 10
```
Ισχύει για γρήγορες εκτιμήσεις. Σφάλμα ±30–40%.

### ⚠️ Σημαντικό
- Ο MQ9 αντιδρά επίσης σε **CH4** και **LPG** — δεν είναι αποκλειστικά για CO
- Απαιτείται **βαθμονόμηση** (R0) σε καθαρό αέρα πριν τη χρήση
- Χρόνος προθέρμανσης **>24 ώρες** για σταθερές ενδείξεις
